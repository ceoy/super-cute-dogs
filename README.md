# Super Cute Dogs

This is a Test-Project for using Vue / Webpack / HTML / CSS etc.

To see the site live visit [https://super-cute-dog.herokuapp.com/](https://super-cute-dog.herokuapp.com)

Images come from [here](https://dog.ceo/dog-api/).

## Run

Run dev-server:
`yarn dev`  

Create build:
`yarn build`
